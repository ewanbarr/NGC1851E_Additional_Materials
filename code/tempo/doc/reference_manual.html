Tempo Reference Manual

This reference manual contains information about Tempo commands, options, file formats, and so forth. It is not intended to be an introduction to Tempo for new users or a treatise on pulsar timing. Rather, it provides experienced Tempo users with a detailed, comprehensive summary of the program.

The document is in ASCII format. It may be downloaded as one long file, or as individual sections.
